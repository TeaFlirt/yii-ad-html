# Чайный сайт - Чайная Лавка "Восточная Мудрость"

Современный веб-сайт чайной лавки с адаптивным дизайном.

## Структура проекта

tea-house-website/
├── index.html # Главная страница
├── about.html # О нас
├── contact.html # Контакты
├── login.html # Вход
├── signup.html # Регистрация
├── welcome.html # Приветствие после регистрации
├── README.md # Документация
└── images/ # Изображения
├── about/ # Изображения для страницы "О нас"
├── team/ # Фото команды
├── gallery/ # Галерея

text


## Особенности

- **Полностью адаптивный дизайн** - работает на всех устройствах
- **Современный UI/UX** с анимациями и интерактивными элементами
- **6 готовых HTML страниц** с полным функционалом
- **Используемые технологии:**
  - HTML5
  - CSS3 (Flexbox, Grid, CSS переменные)
  - Bootstrap 5
  - Font Awesome иконки
  - JavaScript для интерактивности
- **Оптимизированная структура** для быстрой загрузки
